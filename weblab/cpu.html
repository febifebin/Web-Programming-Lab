CPU

The cpu which can be considered the heart of computing system include three maincomponents:

  • control unit
  • Arithemetic logic unit
  • Registers

Control unit

The control unit determines the order in which instructions should be executed and control the retreival of proper operands.
It interprets the instructions of machine.The execution of each instruction is determined by a sequence of control signals produced by control unit

Arithemetic logic unit

ALU performs all matematical and boolean operations

Registers

The registers are temporary storage locations to quickly store and transfer data and instructions being used
